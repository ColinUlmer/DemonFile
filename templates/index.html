DemonFile - Short-term File Hosting

Snapchat, but with files basically

What/Why?

I hate when I have to print a paper out on campus or switch a file between systems by emailing it to myself.

It's a clunky solution. I could put it in my Dropbox but then I forget about it and it sits there and clutters the folder up. Also my sync time is weird and not consistant between computers. Plus if I'm doing this on a campus computer for just a second it's annoying to log into gmail or dropbox on such a public machine, I've seen several people just leave their email/dropbox account open on a lab computer and just leave to print a document without coming back.

Thus DemonFile

DemonFile uses your phone as verification for file upload and download, and we don't even save your phone number to do it. This means you never need to create an account. Once you upload a file you'll have three hours to access it before we delete it. No need to cluter an inbox or Dropbox folder

How To Use

  1. Verify Your Phone Number
  2. Drop file onto screen to upload
  3. Switch systesm, run to the library, etc.
  4. Re enter your phone and code
  5. Download the files you want
  6. Bro down
